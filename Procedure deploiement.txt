4. Configuration de la connexion SQL Server
Dans app.py, vérifiez la fonction get_connection() :

python
def get_connection():
    try:
        conn = pyodbc.connect(
            "Driver={SQL Server};"
            "Server=localhost\\SQLEXPRESS;"  # À adapter
            "Database=REMUCI_VISION;"
            "Trusted_Connection=yes;"
        )
        return conn
    except Exception as e:
        print(f"❌ Erreur de connexion SQL: {e}")
        return None
Adaptez le serveur SQL selon votre environnement :

localhost\\SQLEXPRESS si SQL Express sur le même serveur

IP-DU-SERVEUR-SQL\\NOM_INSTANCE si serveur dédié

7. Configuration réseau
python
# Dans app.py, adapter les CORS pour le réseau :
CORS(app, 
     origins=[
         "http://localhost:5000", 
         "http://192.168.1.67:5000",  # Votre IP actuelle
         "http://IP-DU-SERVEUR:5000", # Nouvelle IP du serveur
         "http://nom-serveur:5000"    # Nom réseau si DNS
     ],
     supports_credentials=True
)
